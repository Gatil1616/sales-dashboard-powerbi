# Sales Performance Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard analyzing sales performance across product categories and time periods.

## Key Metrics
- Total Sales: $4.34M  
- Total Profit: $1.74M  
- Total Orders: 48K  
- Profit Margin: 40%  

## Key Insights
- Sales peaked in 2004 with highest number of orders  
- Classic Cars contribute the highest share of revenue  
- Ferrari 360 Spider is the top-performing product  

## Tools Used
- Power BI  
- Power Query  
- Data Visualization
